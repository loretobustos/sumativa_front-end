☕ Aroma & Código - Cafetería de Especialidad
Aroma & Código es una landing page moderna diseñada para una cafetería de especialidad enfocada en la comunidad de desarrolladores en Maipú, Chile. Este proyecto es el resultado de la Evaluación Sumativa de Front-end, integrando diseño adaptable, animaciones avanzadas y semántica HTML5.

🛠️ Tecnologías y Estándares Aplicados
1. HTML5 Semántico (Estructura)
Uso de etiquetas como <header>, <nav>, <main>, <section>, <article>y <footer>.
Atributos ARIA para mejorar la accesibilidad (WCAG).
Sistema de navegación por anclas con idúnicos.
2. CSS3 Avanzado (Diseño)
Flexbox: Utilizado para la barra de navegación, el carrusel de productos y la sección de contacto.
Animaciones personalizadas: - Loader: Una cafetera animada al inicio de la carga.
Interacción: Granos de café animados cayendo con mix-blend-mode: multiply. Diseño Adaptativo (Responsive): Uso de Media Queries para optimizar la vista en móviles, incluyendo un menú tipo hamburguesa y ajuste dinámico del carrusel con calc().
Variables CSS: Centralización de la paleta de colores para un mantenimiento eficiente.
3. Git y GitHub (Control de versiones)
Gestión de ramas y resolución de conflictos de historial.
Despliegue automatizado mediante GitHub Pages.
